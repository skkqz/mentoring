# Репозиторий обучения стажёра: основы Python

Этот репозиторий создан для систематического обучения стажёра основам языка Python и работе со стандартными библиотеками в рамках стажировки в компании.

## Что здесь есть

* Поэтапный план обучения — от базового синтаксиса до продвинутых возможностей языка.
* Практические задания для закрепления навыков (используются только встроенные возможности Python и стандартная библиотека).
* Примеры кода и решения заданий.

## Как пользоваться

1. Последовательно проходите темы, изучая теорию и примеры кода.
2. Выполняйте практические задания, используя только встроенные типы данных и модули из стандартной библиотеки Python.
3. Сверяйтесь с примерами решений.

Добро пожаловать в команду! Удачи в освоении Python!
