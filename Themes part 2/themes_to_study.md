# Обучение стажёра: Python (Часть 2 — Структуры данных и продвинутые темы)

Этот раздел репозитория посвящён углублённому изучению встроенных структур данных Python, работе с файлами и обработке исключений. Цель — научиться эффективно использовать коллекции, писать надёжный код с обработкой ошибок и работать с внешними данными.

## Темы для изучения

### 2. Базовые коллекции: list (списки) (Теория, практика)
* Что такое список (`list`) и для чего он используется.
* Создание пустого и заполненного списка.
* Индексация элементов (положительные и отрицательные индексы).
* Срезы (`slicing`): синтаксис `[start:stop:step]`, пропуск параметров.
* Операции со списками: конкатенация (`+`), повторение (`*`), проверка вхождения (`in`).
* Длина списка (`len`).
* Вложенные списки и доступ к их элементам.
* Изменяемость списков (отличие от неизменяемых типов).

### 3. Методы для работы со списками (Теория, практика)
* Добавление элементов: `append()`, `extend()`, `insert()`.
* Удаление элементов: `remove()`, `pop()`, `clear()`.
* Поиск и подсчёт: `index()`, `count()`.
* Сортировка: `sort()` и функция `sorted()`.
* Разворот: `reverse()` и срез `[::-1]`.
* Копирование списков: `copy()`, срез `[:]`, функция `list()` (поверхностное копирование).
* Отличие методов, изменяющих список (in-place), от функций, возвращающих новый список.

### 4. List comprehensions (Теория, практика)
* Синтаксис генератора списков: `[выражение for элемент in итерируемый]`.
* Фильтрация с условием `if`.
* Конструкция `if-else` внутри генератора.
* Вложенные циклы в генераторах списков.
* Сравнение с классическим циклом `for` (читаемость, производительность).
* Генераторы множеств (`{}`) и генераторы словарей.
* Разница между генератором списка и выражением-генератором (generator expression).

### 5. Базовые коллекции: строки (Теория, практика)
* Создание строк: одинарные, двойные, тройные кавычки.
* Неизменяемость строк (`imutability`).
* Индексация и срезы строк.
* Конкатенация и повторение строк.
* Экранирование символов.
* Методы строк: `upper()`, `lower()`, `strip()`, `split()`, `join()`, `replace()`, `find()`, `startswith()`, `endswith()`, `isdigit()`, `isalpha()`.
* Форматирование строк: `%`, `format()`, f-строки.
* Строки как последовательности (работа в цикле `for`).

### 6. Базовые коллекции: словари и множества (Теория, практика)

#### Словари (`dict`)
* Что такое словарь: структура «ключ — значение».
* Создание словаря: `{}`, `dict()`.
* Доступ к значениям по ключу: `[]` и метод `get()`.
* Добавление и изменение пар.
* Удаление: `del`, `pop()`, `popitem()`, `clear()`.
* Методы: `keys()`, `values()`, `items()`.
* Проверка наличия ключа (`in`).
* Итерирование по словарю.
* Ограничения на ключи (только неизменяемые типы).

#### Множества (`set`)
* Что такое множество: неупорядоченная коллекция уникальных элементов.
* Создание: `set()`, `{}` (не пустые).
* Добавление: `add()`.
* Удаление: `remove()`, `discard()`, `pop()`, `clear()`.
* Операции: объединение (`|`), пересечение (`&`), разность (`-`), симметрическая разность (`^`).
* Проверка вхождения (`in`).
* Отличие от списков и словарей.
* Замороженные множества (`frozenset`).

### 7. Базовые коллекции: кортежи (Теория, практика)
* Что такое кортеж (`tuple`) и его отличия от списка.
* Создание кортежа: `()`, `tuple()`, запятая для одного элемента.
* Неизменяемость кортежей.
* Индексация и срезы.
* Когда использовать кортежи вместо списков (защита от изменений, производительность).
* Распаковка кортежа (`tuple unpacking`).
* Возврат нескольких значений из функции через кортеж.
* Именованные кортежи (`namedtuple`) — общее представление.

### 8. Функции — Рекурсия (Теория, практика)

#### Рекурсия
* Определение рекурсии.
* Базовый случай (условие выхода).
* Рекурсивный случай (вызов функции из самой себя).
* Стек вызовов и глубина рекурсии.
* Ограничения рекурсии в Python (`RecursionError`, `sys.setrecursionlimit()`).
* Примеры рекурсивных задач (факториал, числа Фибоначчи, обход структур).
* Сравнение рекурсии и итерации (читаемость vs производительность, риск переполнения стека).

### 9. Работа с файлами (Теория, практика)
* Открытие файла: функция `open()`, параметры (имя файла, режим, кодировка).
* Режимы открытия: `'r'` (чтение), `'w'` (запись с перезаписью), `'a'` (дозапись), `'x'` (исключительное создание), `'b'` (бинарный), `'t'` (текстовый).
* Закрытие файла: метод `close()`.
* Контекстный менеджер `with` (автоматическое закрытие).
* Чтение файла: `read()`, `readline()`, `readlines()`, итерация по файлу в цикле.
* Запись в файл: `write()`, `writelines()`.
* Работа с путями: абсолютные и относительные пути, `os.path`, `pathlib`.
* Работа с бинарными файлами (изображения, архивы).
* Работа с JSON: модуль `json`, `dump()`, `load()`, `dumps()`, `loads()`.
* Работа с CSV: модуль `csv`.

### 10. Исключения: работа с ошибками (Теория, практика)
* Что такое исключение (`exception`).
* Иерархия исключений: `BaseException`, `Exception`, конкретные типы (`ValueError`, `TypeError`, `KeyError`, `IndexError`, `FileNotFoundError`, `ZeroDivisionError` и др.).
* Конструкция `try` / `except` / `else` / `finally`.
* Перехват конкретных типов исключений.
* Перехват всех исключений (проблемы такого подхода).
* Получение объекта исключения (`as e`).
* Генерация исключений: `raise`.
* Создание собственных исключений (наследование от `Exception`).
* Проброс исключения (`raise` без аргументов).
* Вложенные блоки `try`.
* Практика: EAFP (`Easier to Ask for Forgiveness than Permission`) vs LBYL (`Look Before You Leap`).
* Исключения в `finally` и их особенности.
